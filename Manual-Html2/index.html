Manual Html para ceporros

1.Iniciar Proyecto

Acceder al bloc de notas

Nota: Para acceder al bloc de notas, tienes que poner la mano en el ratón y hacer click con un dedo al botón derecho del ratón. Una vez que le hayas dado, te apareceda un menú, pon encima suavemente en nuevo y después en documento de texto. Y ahí tendra el bloc de notas.

2.Doctype

Lo primero que hay que poner siempre es entre las etiquetas al lado de la tecla z de la frase que quieras !doctype html

Es importante no olvidarse de poner esa frase, después hay que poner la etiqueta "lang"(Es una etiqueta para poner el lenguaje)

La etiqueta se pone en "las etiquetas al lado de la tecla z" y dentro pondremos "html lang="es"

3.Html lang

Una vez que hayas puesto todo del paso dos

Debajo de la etiqueta html lang, hay que poner html.

No olvides de ponerlos dentro de los "las etiquetas al lado de la tecla z"

4.Head

Ahora pondremos la etiqueta Head

La etiqueta Head significa cabeza, dentro de la cabecera pondremosel titulo y meta charset

El metacharset es el tipo de codificación que usamos en este blog de notas, en este casoUtf-8

Como hemos dicho antes, las etiquetas del head se abren "" y se cierran ""

Nota: si no cierras las etiquetas, no funcionara

5.Metacharset

Como hemos mencionado antes, el metacharset se pone dentro del head y debajo del title

La etiqueta se escribe asi meta chartset="utf-8"

6.Body

Después del head, ahora paseremos albody

Body en español significa cuerpo, dentro del body se pondran todos los elementos de nuestro documento

El body, como hemos dicho las anteriores veces se abre "" y se cierra""

7.Heading

Heading en español significa titulo

Los heading sirven para poner los titulos de los documentos o de los trozos

La etiqueta se abre asi "h" y se cierra asi"/h"

Pero eso no es todo, si queremos poner mas headings hay que poner después de la etiqueta "h1,/h1" y asi hasta seis

Un ejemplo es;

hola esto es

puede que esto sea

una chorrada de ejemplo

8.Ordenerd list

ordenerd list en español significa listas ordenadas

Los ordenerd list como hemos dicho anteriormente, sirven para hacer listas ordenadas

Las etiquetas que debes usar son para empezar "ol", para los elementos de la lista "li" y para cerrar la lista "/ol"

9.Definiton list

definiton list en español significa listas definidas

Su etiqueta es la "dl" para empezar, para los definition term (terminos definidos) se usa la etiqueta "dt" y para los definition data(datos definidos) se usa la etiqueta "dd"

10.Unordered List

Unordered List en español significa listas desordenadas

Los Unordered List como se ha dicho antes en otro punto, sirven para hacer listas desordenadas

Las etiquetas son para empezar la lista "ul", para los elementos igual que en el punto ocho "li" y para cerrar la lista "/ul"

11.Argument

Argument en español significa argumento

Dentro del argumento hay cuatro tipos de cosas dentro

La primera cosa es el "Reverse", un tipo de atributo que no tiene valores y pone la lista empezando por el ultimo, además pone la lista al reves.

La segunda cosa es el "start="numero", una etiqueta que sirve para decir con que número quieres que empiece

La tercera cosa es el "type" significa tipo en español, para decir que tipo de letra quieres

La cuarta cosa es el "li", como hemos dicho anteriorente en el punto "8." sirven para los elementos de lista. pero ahora en el argumento con un "value", significa en español valorar

12.Horizontal rule

Horizontal rule en español significa Línea horizontal

La linea horizontal divide un parrafo de otro

La etiqueta es HR y como hemos dicho anteriormente con "las etiquetas al lado de la tecla z"

Además se pone entremedio de los parrafos normalmente

13.El resaltado

Hay tres tipos de resaltado:

  1. La Italic
  2. La Bold
  3. La underline

La Italic en español significa la "Bastardilla" (para representar palabras extranjeras)

Su etiqueta es la "i" y pone la palabra en cursiva

La Bold en español significa "Negrita"

Su etiqueta es la "b" y pone la palabra en negrita

La Underline en español significa "Subrayada

Su etiqueta es la "u" y pone la palabra subrayada

Stress Emphasis

Sirve para recalcar una parte del texto, le da mas intesidad

Su etiqueta es la "EM"

Strong

Sirve para algo importante, serio o urgente y hace lo mismo que la b, pero son dos cosas distintas

Su etiqueta es "STRONG"

Small

Sirve para representar la letra pequeña

Su etiqueta es "SMALL"

14.Salto de línea

Hay dos tipos de saltos de linea

  1. Break
  2. WORD BREAK

Tanto el break y el word break (oportunidad de salto de linea, si ponemos esta etiqueta delante de una palabra la pondra en otra linea y a veces los ordenadores hacen lo que quieren)

Sus etiquetas son break "BR", word break "WBR"

15.Uniform Resource Locator

Hay varios elementos que son:

  1. Esquema: http,ftp,mailto,ect.
  2. usuario: nombre del usuario en el servidor
  3. contraseña : si se requiere
  4. servidor: nombre de la maquina
  5. puerto: de servicio
  6. localizador: en el servidor (numero del ordenador)

esquema ://usuario:contraseña@servidor:puerto/localizador ( puerto/localizador podria no ponerse o pedirmelo)

16.Hiperenlaces

hay varios tipos de hiperenlaces

  1. Anchor
  2. Href
  3. absoluto
  4. relativo
  5. subdirectorio

El Anchor

Sirve para pillar un atributo que se llama href, sino existe no pasa nada. Se puede marcar o n, el valor del href es la url que quieres ir

Su etiqueta es "a"

El Href

Sirve para marcar un lugar en el que podia haber un enlace si fuera relevante

se escribe : los las etiquetas al lado de la tecla z + a href=

Lo absoluto

se escribe "http://sitio/pagina.html"

El relativo

Sirve para el mismo directorio

Se escribe pagina.html

El Subdirectorio

Se escribe subdirectorio/pagina.html

Directorio anterior

Se escribe ../pagina.html

17.El Hiperenlace interno

Hay tres tipos:

  1. Identificador de elemento
  2. Referencia a elemento
  3. Principio del documento

El identicador de elemento

se escribe con las etiquetas al lado de la tecla z + elto id=eltoX + Elemento X + el cierre con elto

La referencia a elemento

se escribe con las etiquetas al lado de la tecla z + a href=eltoX + Elemento X + el cierre con elto

El principio del documento

se escribe #top

18.Hiperenlace HTML 4

El name es una etiqueta que pone una referencia desde otro sitio

Pero en el html5 hay que usar el #top, actualmente no se usa

se escribe las etiquetas al lado de la tecla z + a name= + nombre del lugar y las comillas

Referencia desde otro documento

se escribe las etiquetas al lado de la tecla z + a href= + URL#identificador + comillas + nombre del lugar (lugar del documento que esta en otro directorio)

Los atributos

Solo si usamos href

hay tres tipos:

  1. download
  2. Target
  3. Otros

Download

Se pone igual y se pone el nombre del destino

El valor opcional: nombre del destino

Target

Se usa el "_blank" y abre una ventana nueva y ahi usa el enlace del href

19.Imagenes

Para poner una imagen hay que poner estos elementos

  1. etiqueta:img +las etiquetas al lado de la tecla z
  2. origen (SouRCe): src
  3. Entre ellos se pone "img src=" + nombre foto + jpg + las etiquetas al lado de la tecla z
  4. texto alternativo : alt
  5. ancho :width
  6. alto: height

Los tipos de imagenes son:

  1. bimap estáticos (con y sin perdida)
  2. PNG (comprimidos y se pueden recuperar), GIF, JPG
  1. Documentos vectoriales
  2. SVG (funciones matematicas,no tiene perdida como los otros)
  1. Animados
  2. APNG( extension del PNG, mostrara la primera imagen aunque no pueda hacer la animación,GIF ( en el fichero se ha metido mas imagenes que crea una animacion)
  1. Otros
  2. PDF (lo soporta en html)
  3. MNG( extesion del PNG para crear animaciones)

20.Mapas de imagen

Sirve para coger una imagen,dividirlo en secciones y se crean enlaces dentro de las secciones y al pincharlo te lleva al contenido del enlace

Se pone "map name=" + mapa + las etiquetas al lado de la tecla z

Independientemente se usa para las imagenes "img usemap=" +#mapa + las etiquetas al lado de la tecla z

Area es un elemento vacio

Los tipos son:

  1. shape : default, rect( x- izq, y-sup, x-dcha, y-inf) circle (x-centro, y-centro, radio) poly ( x1,y1,x2,y2.....,xN,yN)
  2. coords: coordenadas separadas por comas
  3. href: URL asociada al NOHREF
  4. alt : texto alternativo
  5. download
  6. target

21.estructura de map area

Se pone primero "map name

Se pone después "area shape= circle coords= "x,y,r" href=........."

Luego se cierra map

Hay dos tipos de simbolos especiales

  1. formato
  2. Espacio en blanco

El formato

y nombre_del_simbolo;(y mas el nombre del simbolo y ;)

El espacio en blanco

Se escribe Non-Breaking Space:nbsp (espacio sin romperlo, cuando quieras poner dos palabras sin romper y que esten en la linea se pone nsbp. Se pone para poner espacios en blanco y que no haga salto de linea) Eñe: ñ Ñ Acentos agudos: á Á é ...

23.Tablas

Se escribe primero "Table" para la tabla

Luego se escribe"Table row" para las filas

Después se escribe "td" para las columnas

Y se escribe "th" para el encabezado

Otros elementos para poner en la tabla son:

  1. thead para el encabezado de la tabla
  2. tbody para el cuerpo de la tabla
  3. tfoot para el sumario de la tabla
  4. caption para la leyenda
